# Markdown Table

> Format output as a markdown table

## Snippet

```
Present the results as a markdown table.
```

## Usage

Use when you need:
- Easy comparison of options
- Structured data presentation
- Quick scanning of information

## Examples

### Comparison
```
Compare these frameworks.

Present the results as a markdown table with columns: Framework, Pros, Cons, Best For.
```

### Audit Results
```
Audit this code for issues.

Present the results as a markdown table with columns: Issue, Location, Severity, Fix.
```

## Variations

### With Specific Columns
```
Present as a table with columns: [Column1], [Column2], [Column3].
```

### Sortable
```
Present as a table sorted by [column] descending.
```
