# Checklist

> Format output as actionable checklist

## Snippet

```
Present the results as a checklist I can work through.
```

## Usage

Use when:
- Need actionable items
- Tracking progress
- Ensuring nothing is missed

## Examples

### Pre-Launch
```
What should I check before deploying?

Present the results as a checklist I can work through.
```

### Code Review
```
Review this code.

Present issues as a checklist of fixes needed.
```

## Output Format

```markdown
## Checklist

- [ ] Item 1
- [ ] Item 2
  - [ ] Sub-item 2a
  - [ ] Sub-item 2b
- [ ] Item 3
```

## Variations

### Prioritized
```
Present as a prioritized checklist. Most important first.
```

### Categorized
```
Present as a checklist grouped by category.
```

### With Notes
```
Present as a checklist with brief notes on each item.
```
