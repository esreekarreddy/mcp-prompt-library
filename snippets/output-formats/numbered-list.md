# Numbered List

> Format output as numbered list

## Snippet

```
Present as a numbered list in order of priority/importance.
```

## Usage

Use when:
- Order matters
- Sequence of steps
- Ranked recommendations

## Examples

### Steps
```
How do I deploy this?

Present as a numbered list of steps.
```

### Priorities
```
What should I focus on?

Present as a numbered list in order of priority.
```

## Variations

### With Details
```
Present as a numbered list. For each item, include a brief explanation.
```

### Grouped
```
Present as numbered lists grouped by category.
```
