# Ultrathink

> Force deeper analysis and reasoning

## Snippet

```
Ultrathink.
```

## Usage

Append to any prompt when you need:
- Deeper analysis of complex problems
- More thorough exploration of edge cases
- Higher quality reasoning
- Consideration of non-obvious factors

## Examples

### With Debugger
```
I'm stuck in a debugging loop. The bug: [describe it]
[...full debugger prompt...]

Ultrathink.
```

### With Security Audit
```
Security audit before production.
[...full security audit prompt...]

Ultrathink.
```

### With Architecture Analysis
```
Analyze this codebase and give me:
[...full architecture prompt...]

Ultrathink.
```

## When to Use

| Situation | Use Ultrathink? |
|-----------|-----------------|
| Simple code changes | No |
| Complex debugging | Yes |
| Security audits | Yes |
| Architecture decisions | Yes |
| Performance optimization | Yes |
| Quick questions | No |
| Code review | Maybe |

## Variations

### Extended Thinking
```
Think deeply about this. Take your time. Consider edge cases.
```

### Explicit Reasoning
```
Think step by step. Show your reasoning before concluding.
```
