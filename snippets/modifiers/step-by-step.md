# Step by Step

> Request methodical, incremental approach

## Snippet

```
Work through this step by step. Show each step before moving to the next.
```

## Usage

Use when you need:
- Clear visibility into the process
- Ability to catch errors early
- Easier debugging of AI reasoning
- Learning from the approach

## Examples

### With Implementation
```
Implement this feature.

Work through this step by step. Show each step before moving to the next.
```

### With Problem Solving
```
How do I optimize this database query?

Work through this step by step. Show each step before moving to the next.
```

## Variations

### Numbered Steps
```
Break this into numbered steps. Complete each step before moving to the next.
```

### With Verification
```
Work step by step. After each step, verify it's correct before proceeding.
```

### Pause Points
```
Work step by step. Pause after each major step for my approval before continuing.
```
